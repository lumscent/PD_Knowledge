# PD Knowledge — MyLink Design Team

Репозиторий с инструментами и знаниями дизайн-команды MyLink.
Подключается к Claude Desktop и даёт доступ к скиллам прямо в чате.

---

## Скиллы

### 🔍 design-audit
Аудит интерфейсов и UX-копирайтинга. Принимает скриншоты или ссылку на Figma.
При проверке сверяется с библиотекой компонентов в [Storybook](https://storybook.mylink.kz).

**Команды:**

| Команда | Что делает |
|---|---|
| `/audit [продукт]` | Полный UX/UI аудит экрана или продукта |
| `/copy-review [экран]` | Аудит UX-копирайтинга — кнопки, заголовки, ошибки, подсказки |

**Как использовать:**
1. Открой чат в Claude Desktop
2. Выбери скилл `design-audit` через меню `/`
3. Введи команду и приложи скриншоты или ссылку на Figma
4. Claude проанализирует по чеклисту и предложит создать задачи в Jira

---

### 🎫 jira-task-maker
Создание структурированных задач в Jira прямо из чата.
Работает в два шага: сначала показывает черновик, потом создаёт задачу после подтверждения.

**Команды:**

| Команда | Что делает |
|---|---|
| `/ux-task [экран] [проблема]` | Создаёт UX-задачу по шаблону |
| `/tech-debt [компонент] [описание]` | Заводит технический долг по дизайну |

**Как использовать:**
1. Выбери скилл `jira-task-maker` через меню `/`
2. Введи команду с описанием проблемы
3. Проверь черновик задачи
4. Подтверди — Claude создаст тикет и вернёт ссылку

**Примеры:**
```
/ux-task Профиль Кнопки действий не имеют визуальной иерархии
/tech-debt Иконки Смешаны stroke и filled стили в сайдбаре
```

---

## Типичный флоу

```
/audit → найдены проблемы → /ux-task или /tech-debt → тикет в Jira
```

1. Запускаешь аудит через `design-audit`
2. По итогам аудита переключаешься на `jira-task-maker`
3. Создаёшь задачи по найденным проблемам

## Структура репо

```
skills/
  design-audit/          — скилл для аудита
    SKILL.md
    references/
      checklist.md       — чеклист аудита по 4 направлениям
      severity.md        — градация критичности (Critical / High / Medium / Low)
      report-format.md   — формат итогового отчёта
      copy-checklist.md  — чеклист для /copy-review

  jira-task-maker/       — скилл для создания задач
    SKILL.md
    references/
      ux-task.md         — шаблон UX-задачи
      tech-debt.md       — шаблон технического долга
      jira-config.md     — конфиг Jira: лейблы, приоритеты, исполнители
```

---

## Настройка

1. Установи [Claude Desktop](https://claude.ai/download)
2. Открой раздел **Skills** в настройках
3. Добавь скиллы из папки `skills/` этого репо
4. Подключи коннекторы **Jira** и **Figma** в разделе **Connectors**
