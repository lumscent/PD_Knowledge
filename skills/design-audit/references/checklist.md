# Чеклист аудита — Design Audit

Используется при `/audit`. Проверяй по всем 4 направлениям.

---

## 1. Визуальная консистентность

### Токены и стили
- [ ] Цвета соответствуют палитре из Storybook
- [ ] Типографика — только размеры и начертания из шкалы
- [ ] Отступы кратны шкале токенов
- [ ] Радиусы углов соответствуют токенам
- [ ] Тени используются по правилам

### Компоненты
- [ ] Компоненты из дизайн-системы, не кастомные
- [ ] Нет визуальных отклонений от эталона в Storybook
- [ ] Figma-макет соответствует продакшну

### Иконки
- [ ] Единый стиль иконок (не смешаны stroke и filled)
- [ ] Размеры иконок соответствуют токенам
- [ ] Иконки из утверждённой библиотеки

---

## 2. UX и usability

### Навигация и структура
- [ ] Иерархия информации читается с первого взгляда
- [ ] Навигация понятна — пользователь знает где он находится
- [ ] Основное действие на экране визуально выделено
- [ ] Не более одной primary-кнопки на экране

### User flows
- [ ] Ключевые сценарии пройдены как пользователь
- [ ] Нет тупиков — всегда есть следующий шаг
- [ ] Количество шагов до цели минимально

### Состояния
- [ ] Loading — есть и понятен
- [ ] Empty state — есть объяснение и CTA
- [ ] Error — понятное сообщение и способ исправить
- [ ] Disabled — понятно почему недоступно
- [ ] Success — подтверждение что действие выполнено

### Мобильная адаптация
- [ ] Все элементы доступны на мобильном
- [ ] Touch-зоны не меньше 44px
- [ ] Контент не обрезается

---

## 3. Компоненты

- [ ] Все состояния реализованы (hover, active, focused, disabled, error)
- [ ] Компоненты консистентны между экранами
- [ ] Интерактивные элементы визуально отличимы от статичных
- [ ] Формы: лейблы, плейсхолдеры, валидация — по правилам

---

## 4. UX-копирайтинг

- [ ] Кнопки — глагол действия, не "ОК" и не "Да"
- [ ] Заголовки — понятны без контекста
- [ ] Ошибки — объясняют причину и что делать
- [ ] Пустые состояния — объясняют что происходит и куда идти
- [ ] Нет опечаток и грамматических ошибок
- [ ] Единый tone of voice
- [ ] Единое обращение к пользователю (ты/вы)
