# Шаблон: /design (Design System)

**Проект:** MLDS
**Префикс названия:** `Design: ...`

## Название задачи
`Design: {Что и где меняется}`

Примеры:
- `Design: Кнопки — добавить состояние loading`
- `Design: Tabs — расширить вариант Segmented`
- `Design: Обновить токены нейтральной шкалы`
- `Design: Иконочная библиотека — унифицировать stroke и filled`
