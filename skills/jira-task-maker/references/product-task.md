# Шаблон: /product

**Префикс названия:** `[Продукт] ...`

## Название задачи
`[{Продукт}] {Экран — краткая суть}`

Примеры:
- `[MyTeam] Профиль — добавить иерархию у блока контактов`
- `[MyLink] Hiring pipeline — пустое состояние воронки`
- `[MyTeam] Дашборд — убрать личные данные из превью`
