# Шаблон: /board

**Проект:** DML
**Префикс названия:** `Действие: ...`

## Допустимые действия
- **Аудит** — проверка существующего продукта по чеклисту
- **Бенчмарк** — изучение решений конкретных фич в других продуктах
- **Ресерч** — исследование пользователей, сценариев, паттернов
- **Анализ** — разбор конкретного вопроса или данных
- **Конкурентный обзор** — сравнение с конкурентами

## Название задачи
`{Действие}: {Тема}`

Примеры:
- `Аудит: UX/UI продукта MyTeam`
- `Бенчмарк: онбординг в HR-системах`
- `Ресерч: сценарии работы рекрутера в ATS`
- `Анализ: использование компонента Chips`
- `Конкурентный обзор: ATS-платформы в РФ и КЗ`
