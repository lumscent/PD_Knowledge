# Jira Task Maker — MyLink

Скилл для быстрого создания структурированных задач в Jira.
Всегда работает в два шага: сначала черновик, потом создание после подтверждения.

## Конфигурация Jira

- **cloudId**: `bf61b230-67a7-4150-82eb-4981e9ddfd51`
- **Проект**: `DML`
- **Тип задачи**: `Задача`
- **Designer approve**: всегда `{ "id": "10345" }`
- **Исполнитель по умолчанию**: Assema Muzapbarova `712020:0c45ea9c-1cf6-4ac2-9f89-3f17d187493d`

## Команды

### /ux-task [экран] [проблема]
Оформляет UX-проблему как задачу в Jira.

**Процесс:**
1. Проанализировать проблему
2. Задать уточняющие вопросы если описание неполное
3. Сформировать черновик по шаблону из [ux-task.md](references/ux-task.md)
4. Показать дизайнеру для проверки
5. После подтверждения — создать задачу в Jira

---

### /tech-debt [компонент] [описание]
Заводит технический долг по дизайну.

**Процесс:**
1. Оформить по шаблону из [tech-debt.md](references/tech-debt.md)
2. Автоматически поставить лейбл `tech-debt`
3. Показать черновик дизайнеру
4. После подтверждения — создать задачу в Jira

---

## Правило

Без подтверждения дизайнера задача **не создаётся**.
